# Network Scanning Project

## Project Overview
This project demonstrates network scanning using Nmap and packet analysis using Wireshark. The objective was to identify open and closed ports, capture network packets, and analyse TCP communication.

## Tools Used
- Nmap
- Wireshark
- macOS

## Project Structure

```
Network_Scanning_Project/
├── Phase 1/
├── Phase 2/
├── Phase 3/
├── Report/
└── README.md
```

## Features
- Port scanning using Nmap
- Packet capture using Wireshark
- TCP three-way handshake analysis
- Open and closed port identification
- Protocol hierarchy analysis
- Network conversation analysis
